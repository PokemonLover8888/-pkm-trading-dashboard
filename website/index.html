Tutorial

How to Trade

Get started in minutes

1

Join Our Discord

Join the PKM-Universe Discord server to access all trading features.

Join Server
2

Use /pkhex Command

In any trade channel, use /pkhex create or /pkhex quick to create your Pokemon.

/pkhex create pokemon:Charizard shiny:True
3

Enter Link Code

The bot will give you a Link Trade code. Open your game and start a Link Trade with that code.

Make sure you're connected to the internet in-game!
4

Complete Trade

Trade any Pokemon to the bot and receive your custom Pokemon! The trade completes in seconds.

That's it! Enjoy your new Pokemon!
